How do I see the differences between HTML and CSS

  1. What is the difference between the usage of HTML and CSS?

    2. HTML is used to structure the the content on the web page, while CSS is used for styling the visual presentation of the web page.

  2. What is the newest version of HTML?

    3. HTML5

  3. What are the top five most used HTML5 elements? If not sure, make a guess.
    1. <div>
    2. <p>
    3. <h1> to <h6>
    4. <a>
    5. <img>
  4. Have you check out the HTML5 cheatsheet?

    5. I have, it can be found here.

  5. What is the difference between semantic and non-semantic elements?

    6. Semantic elements convey information about the type of content they contain to both the browser and the developer. Where as non-semantic elements do not define any meaning about their content and are used purely for styling purposes or as containers.

  6. Why do we prefer using semantic elements?

    7. We prefer to use semantic elements for their ability to provide clearer structure and to maintain consistency across web pages.

  7. Do all browsers support all the newest HTML/CSS elements?

    8. No, not all browsers support all the latest HTML and CSS elements. It largely depends on: browser version, browser vendor and the rendering engine. An example of a browsers that does not support the latest elements is Internet Explorer.

  8. Test the following elements on Can I Use: svg, nav, code, form, datalist. Which browser version supports them and which do not?

    • Older versions of Internet Explorer and Android Browser do not support svg.

    • Older versions of most browsers do not support nav.

    • Older versions of Opera, Opera Mobile and Android Browser do not support code.

    • Older versions of most major browsers do not support form, but Internet Explorer does not support it at all

    • Older versions of most major browsers do not support datalist, but Opera Mini, Firefox for Android and KaiOS do not support it at all.
